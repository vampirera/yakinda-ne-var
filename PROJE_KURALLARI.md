Proje: Yakında Ne Var? — Türkçe yerel pazar ve teslimat uygulaması (Marmaris/Muğla, Türkiye)

TECH STACK:
- Backend: Node.js + Express → Railway'de canlı
- Frontend: Sade HTML/CSS/JS → GitHub Pages
- Veritabanı: PostgreSQL (Railway)
- Resimler: Cloudinary
- Harita: Leaflet.js
- Bildirim: WhatsApp (wa.me linkleri)

BACKEND URL: https://yakinda-ne-var-backend-production.up.railway.app
Admin girişi: ADMIN_TELEFON ve ADMIN_SIFRE env variable'larıyla çalışıyor.
